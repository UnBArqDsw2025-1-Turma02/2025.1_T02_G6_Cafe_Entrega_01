# Léxico
Léxico é um artefato utilizado na Engenharia de Requisitos para definir e padronizar os termos e expressões usados dentro do contexto de um sistema de software. Ele consiste em uma lista estruturada de palavras, frases ou conceitos relevantes, cada um acompanhado de uma descrição clara e concisa, explicando seu significado no escopo do projeto.
 Esse instrumento ajuda a garantir o alinhamento entre stakeholders, analistas e desenvolvedores, reduzindo ambiguidades e facilitando a comunicação.
 	
## Estrutura do Léxico

### Símbolo:
 Representa o termo ou expressão que será definido no léxico. É o nome de um conceito, ação ou entidade relevante para o sistema, e serve como ponto central para a definição.


### Classificação:
 Indica a natureza do símbolo, categorizando-o conforme seu papel no sistema. Pode incluir:
 - Verbo: Ações ou operações realizadas pelo sistema ou por seus usuários.


 - Objeto: Elementos manipulados ou representados no sistema, tangíveis ou conceituais.


 - Sujeito: Agentes que interagem com o sistema (usuários, atores externos).


 - Evento: Ocorrências importantes que alteram ou influenciam o funcionamento do sistema.


 - Estado: Condições possíveis de um objeto ou do sistema em determinado momento.


 - Atributo: Características ou propriedades associadas a objetos ou eventos.


 - Lugar: Locais relevantes no domínio que influenciam operações ou fluxo de informação.


 - Sinônimo:
 Outras palavras ou expressões que podem ser usadas como alternativas ao símbolo principal. Isso ajuda a identificar variações de vocabulário e garante a padronização dos termos.


### Noção:
 - Fornece uma explicação objetiva e contextualizada do símbolo. A definição deve ser clara, concisa e compreensível, refletindo o significado real do termo dentro do escopo do sistema.


### Impacto:
 - Descreve como o símbolo afeta ou é afetado pelo sistema. Essa seção destaca o papel funcional ou estrutural do termo, suas interações com outros elementos e sua relevância para o comportamento ou para os dados do sistema.



## Verbos

| Elemento   | Descrição |
|------------|-----------|
| **Curtir** | Marcação de um tópico como preferido por um usuário. <br>**Impacto**: Melhora ranqueamento do conteúdo e cria lista personalizada. |
| **Seguir** | Marcação de um usuário como referência para acompanhar postagens. <br>**Impacto**: Fortalece conexões e relevância do perfil seguido. |
| **Cadastro** | Ação de criar uma nova conta na plataforma. <br>**Impacto**: Etapa inicial de entrada no sistema. |
| **Login** | Processo de entrada no sistema com credenciais válidas. <br>**Impacto**: Permite acesso a recursos restritos. |
| **Confirmação de Cadastro** | Ação de validar o e-mail do usuário para ativar sua conta. <br>**Impacto**: Ajuda a evitar contas falsas. |
| **Filtro de Tópico** | Ação de limitar os tópicos exibidos com base em critérios. <br>**Impacto**: Melhora a experiência de navegação. |
| **Busca de Tópico** | Ação de encontrar tópicos com base em palavras-chave. <br>**Impacto**: Recurso essencial para descoberta de conteúdo. |
| **Avaliação de Cafeteria** | Ação de atribuir nota e comentário a uma cafeteria. <br>**Impacto**: Influencia rankings e reputação de estabelecimentos. |
| **Ordenação de Avaliações** | Ação de definir a ordem das avaliações (mais recentes, melhor avaliadas). <br>**Impacto**: Melhora análise de feedback. |
| **Compartilhamento** | Ação de copiar ou distribuir um link de conteúdo. <br>**Impacto**: Amplia o alcance da plataforma. |

## Objetos

| Elemento   | Descrição |
|------------|-----------|
| **Fórum** | Espaço digital onde os usuários interagem por meio de tópicos, comentários e votos sobre café. <br>**Impacto**: Estrutura central do sistema. |
| **Tópico** | Unidade de conteúdo criada por um usuário. <br>**Impacto**: Pode ser avaliado, favoritado, comentado e ranqueado. |
| **Evento** | Atividade agendada envolvendo a comunidade. <br>**Impacto**: Usuários e cafeterias podem organizar e promover. |
| **Ranking de Usuário** | Lista que ordena os usuários com base em interações. <br>**Impacto**: Gera engajamento e senso de comunidade. |
| **Ranking de Cafeteria** | Avaliação comparativa entre cafeterias. <br>**Impacto**: Referência para visitação. |
| **Ranking de Post** | Lista dos tópicos mais curtidos ou comentados. <br>**Impacto**: Influencia o destaque na página inicial. |
| **Usuário** | Pessoa cadastrada na plataforma. <br>**Impacto**: Ator principal da plataforma. |
| **Barista** | Usuário que trabalha com preparo de café profissionalmente. Pode ter selo de verificação. <br>**Impacto**: Pode ter permissões para cadastrar receitas e ajustar equipamentos. |
| **Cafeteria** | Perfil institucional de um estabelecimento. <br>**Impacto**: Pode postar conteúdo, divulgar eventos, ser avaliada e ranqueada. |
| **Produto** | Bem de consumo relacionado ao café. <br>**Impacto**: Pode ser divulgado em tópicos, avaliado, comentado. |
| **Comentário** | Resposta de um usuário a um tópico. <br>**Impacto**: Promove interação entre membros. |
| **E-mail** | Informação usada como identificação de usuário e para comunicação. <br>**Impacto**: Usado no login, cadastro e notificações. |
| **Senha** | Código secreto usado para autenticação. <br>**Impacto**: Deve ser armazenada de forma segura. |
| **Notícia Completa** | Conteúdo estendido de uma notícia após ser clicada. <br>**Impacto**: Aprofunda a informação oferecida na listagem. |
| **SCA** | Associação que define padrões de qualidade para café. <br>**Impacto**: Pode fornecer dados e critérios usados no sistema. |

## Estados

| Elemento   | Descrição |
|------------|-----------|
| **Receita** | Conteúdo que ensina o preparo de bebidas ou comidas à base de café. <br>**Impacto**: Influencia engajamento e ranqueamento. |
| **Dica** | Sugestão prática para melhorar a experiência com o café. <br>**Impacto**: Pode aumentar reputação do autor. |
| **Avaliação** | Julgamento feito por um usuário sobre um conteúdo ou perfil. <br>**Impacto**: Influencia rankings. |
| **Notícia** | Informação sobre eventos, lançamentos, concursos ou tendências. <br>**Impacto**: Pode ser publicada por perfis verificados. |
| **Sessão** | Estado que mantém o usuário conectado após login. <br>**Impacto**: Essencial para navegação contínua. |
| **Média de Avaliações** | Representação da média das avaliações de uma cafeteria. <br>**Impacto**: Ajuda usuários a comparar cafeterias. |

## Atributos

| Elemento   | Descrição |
|------------|-----------|
| **Torra** | Nível de aquecimento aplicado ao grão verde para desenvolver sabor. <br>**Impacto**: Pode ser mencionado como parte de uma descrição ou observação. Não será cadastrada formalmente. |
| **Moagem** | Grau de trituração do grão de café. <br>**Impacto**: Usada em parâmetros de preparo. |
| **Perfil Sensorial** | Conjunto de características perceptíveis no café (frutado, floral, etc.). <br>**Impacto**: Pode ser exibido em produtos, ajudar em recomendações e classificação. |
| **Acidez** | Sensação gustativa associada a frutas e frescor. <br>**Impacto**: Pode ser campo em ficha de produto ou critério de avaliação. |
| **Café Especial** | Café com pontuação acima de 80 pela SCA. <br>**Impacto**: Requer rastreabilidade, controle de qualidade e certificações. |
